# 2026-04-18 Gapyeong Camping Plan

## 1. 요약

- 봄철 일교차가 큰 1박 2일 가족 캠핑이다.
- 낮 활동은 편하지만 밤 기온이 낮아 보온 준비가 필요하다.
- 유치원생 동행 기준으로 여벌 옷, 보온 외투, 비상약 계열 준비가 중요하다.

## 2. 추천 장비

- 4인용 터널 텐트 1동
  - 보유한 두 종류 중 방수성과 패밀리 사용성이 높은 모델 우선
- 3계절 침낭 2개
  - 성인용 1개
  - 키즈용 1개
- LED 랜턴 2개
- 화로대
- 부탄 스토브
- 접이식 테이블

## 3. 연령대별 개인 준비물

### 성인

- 썬글라스
- 썬크림
- 바람막이 자켓
- 저녁용 보온 외투
- 개인 세면도구

### 유치원생

- 여벌 옷 2세트
- 체온 조절용 겉옷
- 얇은 모자
- 개인 물병
- 소아용 해열제 및 기본 비상약
- 피부 민감 대비 보습용품

## 4. 부족한 장비/소모품

- 모기향 재고가 낮아 추가 구매 권장
- 랜턴 배터리 상태 점검 필요
- 차량 적재 상태 사전 확인 필요

## 5. 출발 전 체크리스트

- 랜턴 충전 확인
- 보조배터리 충전 확인
- 유치원생 여벌 옷 준비
- 밤 기온 대비 보온 외투 확인
- 부탄가스 수량 확인

## 6. 추천 식단/요리

- 저녁: 바비큐 + 어묵탕
- 아침: 샌드위치 + 컵스프

추천 이유:

- 저녁에는 바깥 활동 후 따뜻한 국물 메뉴가 체감 만족도를 높인다.
- 아침은 간단하고 정리 부담이 적은 메뉴가 적합하다.

## 7. 이동 중 추천 장소

- 서울 출발 후 가평 진입 전 강변 산책 포인트 1곳
- 점심 식사가 가능한 현지 식당 1곳

## 8. 캠핑장 주변 추천

- 산책 가능한 강변 코스
- 간단 장보기가 가능한 마트
- 아이와 함께 쉬기 좋은 카페

## 9. 주의사항 및 한계

- 이 결과는 trip 파일에 입력된 수동 날씨 정보를 기준으로 작성되었다.
- 실제 우천 가능성이 생기면 우비, 방수 수납, 실내 대체 코스를 추가 검토해야 한다.
